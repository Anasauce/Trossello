# Trossello

Open Source Trello Clone


## Contributing

1. Fork this repository
2. Take a ticket from the [trello board](https://trello.com/b/QIKeSwK0/trossello-oss-trello)
3. Cut a topic branch
4. Finish the ticket
5. Submit a pull request


## Architecture

- Node
- Express
- Webpack
- Babel es2016
- React
- Redux
